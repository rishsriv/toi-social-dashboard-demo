# Times of India Facebook Engagement Dashboard

An interactive React dashboard for analyzing Facebook engagement data from Times of India posts.

## Features

- **Dark Mode**: Toggle between light and dark themes
- **Export Options**: Download filtered data as CSV or capture dashboard as PNG
- **Advanced Filtering**: Filter posts by title or message content
- **Engagement Visualization**: View top posts engagement and distribution
- **Responsive Design**: Works on all screen sizes

## Charts and Analytics

- **Top Posts Chart**: Bar chart showing likes, comments, and shares for top 10 posts
- **Engagement Distribution**: Pie chart showing the proportion of likes, comments, and shares
- **Detailed Data Table**: Sortable and paginated table with all post details

## Getting Started

### Prerequisites

- Node.js (version 14 or above)
- npm or yarn

### Installation

1. Clone the repository
2. Navigate to the project directory:
   ```
   cd dashboard
   ```
3. Install dependencies:
   ```
   npm install
   ```
   or
   ```
   yarn install
   ```

### Running the Application

Start the development server:
```
npm start
```
or
```
yarn start
```

The application will be available at [http://localhost:3000](http://localhost:3000).

## Data Source

The dashboard uses Facebook engagement data from Times of India posts, with metrics including:
- Post message and title
- Likes, comments, and shares counts
- Audience demographics
- Engagement metrics
